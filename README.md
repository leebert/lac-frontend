# lac-frontend
Frontend for LAC prototype. Backend is [here](https://github.com/leebert/lac-backend).

## Setup
`npm install`

### Access Dev Backend
1. Start backend (in `lac-backend/`): `npm start` (runs on port 8080)
2. Start frontend `npm run dev:local`

### Access Production Backend
`npm run dev:cloud` - Connects to production Google Cloud backend
